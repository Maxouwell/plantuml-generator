= Description
// tag::description[]

With the help of this configuration parameter you are able to 
add any PlantUML configuration or even normal text to the beginning
of the generated PlantUML diagram.
The parameter is a list of strings and every string is a new line in the
PlantUML diagram.

If this parameter is not added or if the parameter is an empty list, 
nothing is added to the diagram.

// end::description[]

= Default value
// tag::defaultvalue[]
The default value for this configuration is an empty list.
// end::defaultvalue[]
