= Description
// tag::description[]
With this configuration toggle you can remove all methods 
from the generated diagram text.

If you want to remove specific methods from the diagram, 
you have to use the
xref:./method-blacklist-regexp.adoc[methods blacklist regular expression]
configuration option.

If you want to hide all methods on the rendered diagram but do not want to 
remove them from the diagram text, you have to use the
xref:./hide-methods.adoc[hide methods]
configuration option.
// end::description[]

= Default value
// tag::defaultvalue[]
The default value of this configuration option is false (nothing is removed).
// end::defaultvalue[]
