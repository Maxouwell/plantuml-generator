= Description
// tag::description[]
With this configuration toggle you can hide all methods in the rendered diagram
but they are still part of the generated diagram text.

If you want to remove specific methods from the diagram, 
you have to use the
xref:./method-blacklist-regexp.adoc[method blacklist regular expression]
configuration option.

If you want to remove all methods from the diagram completely, you have 
to use the
xref:./remove-methods.adoc[remove methods] configuration option.
// end::description[]


= Default value
// tag::defaultvalue[]
The default value of this configuration option is *false* (nothing is hidden).
// end::defaultvalue[]
